#### 使用说明

1.  .config
    + Unix系统生态工具统一配置目录
    + `用户根目录/.config`
2.  UltiSnips
    + 默认读取vim主目录
    + `用户根目录/.vim/UltiSnips`
3.  ideavim
    + Unix或Windows都在用户根目录
    + Unix
        + `用户根目录/.ideavimrc`
    + Windows
        + `C:\Users\用户名\.ideavimrc`
4.  tmux
    + 两个配置文件都需要放在用户根目录
    + `用户根目录/.tmux.conf`
    + `用户根目录/.tmux.conf.local`
5.  幻灯片
    + Mac用户可以使用 `Vim入门详解-open.key`
    + Windows用户可以使用 `Vim入门详解-open.pptx`
6.  思维导图
    + Mac用户可以使用 `Vim入门.mindnode`
    + Windows用户可以使用 `Vim入门.xmind`
